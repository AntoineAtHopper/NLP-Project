# NLP Project

## Setup

```
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Usage

### Interpreter mode
```
python main.py
```

### Evaluation mode
```
python evaluate.py
```

## Project structure
```
/main.py
/evaluate.py
/questions_answering.py
/searchable_index.py
/resources
```
